💻 SOURCE CODE FILES:
═════════════════════════════════════════════════════════════════════════════

1. 🐍 train_model.py (Backend - ML Model Training)
   ├─ Purpose: Generate data & train model
   ├─ Run: python train_model.py
   ├─ Does:
   │  ├─ Generates 250 synthetic records
   │  ├─ Tamil Nadu (Kanyakumari) specific data
   │  ├─ 5 bus routes + 15 bus stops
   │  ├─ Preprocesses data
   │  ├─ Trains Linear Regression
   │  ├─ Evaluates model performance
   │  ├─ Saves model to models/bus_model.joblib
   │  └─ Saves encoders to models/encoders.joblib
   ├─ Output: Shows performance metrics
   ├─ Time: ~1-2 seconds
   ├─ Required: Run only ONCE before starting app.py
   └─ Lines: 600+

2. 🌐 app.py (Backend - Flask REST API)
   ├─ Purpose: Run Flask web server
   ├─ Run: python app.py
   ├─ Does:
   │  ├─ Loads pre-trained model
   │  ├─ Serves HTML page
   │  ├─ Provides /predict API endpoint
   │  ├─ Validates inputs
   │  ├─ Makes predictions
   │  └─ Returns JSON responses
   ├─ API: POST /predict
   ├─ Server: http://localhost:5000
   ├─ Time: Starts in <5 seconds
   ├─ Keep running during: Web usage
   └─ Lines: 120+
